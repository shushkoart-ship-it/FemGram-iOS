# FemGram — Telegram Client with Superpowers

> **Your chat. Your rules. No limits.**

FemGram is a powerful, feature-packed fork of Telegram for iOS, built on top of the official open-source client. It combines all the advanced features of AyuGram with exclusive FemGram tweaks — giving you complete control over your messaging experience.

---

## ✨ Key Features

### 👻 Ghost Mode
- Read messages without sending read receipts
- View stories without marking them as seen
- Stay offline while using the app
- Hide typing status
- Auto-offline mode
- Scheduled message sending
- Story opening warnings

### 💾 Message Saving
- Automatically save deleted messages
- Save edited message history
- Auto-download media before it expires
- View deleted messages in a separate chat
- Export chats to HTML/TXT with media

### 🧹 Smart Filters
- Hide messages by regex rules
- Global and per-chat filters
- Block messages from blocked users
- Import/export filter lists

### 🔄 AyuForward
- Forward messages from no-forward channels
- Auto-download media
- Forward without author attribution

### 👑 Local Telegram Premium
- All premium features — no subscription needed
- Custom emoji status
- Profile color & icon

### 🎨 Custom Themes
- Pre-made themes: Red, Blue, Green, Purple, Pink, Orange, Dark, Light
- Fully customizable colors
- Save your own themes
- Import/export themes

### 🖋 Custom Fonts
- Change interface font
- Change message font
- Monospace font selection
- Font size control
- Import custom `.ttf` / `.otf` fonts

### 🛠️ FemGram Exclusive Tweaks (17+)
- Auto-reply by keywords
- Hidden chats (with swipe-to-hide)
- IP whitelist for access control
- Custom swipe gestures
- Built-in browser with ad blocking
- Auto-cleanup chats by rules
- Mass participant export
- Emoji replacement
- Disable stickers/GIF animations
- Call auto-responder
- And much more...

### 📦 Extra AyuGram Features
- Peek Online
- Message Shot (screenshot creator)
- Streamer Mode
- Disable similar channels
- Custom context menu
- Application icon picker
- And 30+ more...

## 🚀 Getting Started

### Prerequisites

- macOS with Xcode 14+
- iOS 15+ device or simulator
- Apple ID (free for development)

### Build from Source

```bash
# Clone the repository
git clone https://github.com/yourusername/FemGram.git
cd FemGram

# Open the project
open FemGram.xcworkspace

# Build and run (Cmd + R)
```

Installation on iPhone

1. Open the project in Xcode
2. Select your device or simulator
3. Press Run ▶️
4. FemGram will be installed on your device
